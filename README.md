## Contributions

### After cloning the repo, run

Install node modules

```npm
npm install
```

Start development server

```npm
npm start
```

## API

Movies App uses omdbapi to show the data.

Get your api key from [omdbapi](https://omdbapi.com/apikey.aspx)
